dsPrint 1.0
Windows 10 (64x)
1.Установите Python: https://www.python.org/downloads/ -> latest version (настройки setup - стандартные).
2.Включите Bluetooth.
3.Включите принтер niimbot D11.
4.Подключите принтер к компьютеру: Параметры -> Устройства -> Bluetooth и другие устройства -> Добавление Bluetooth или другого устройства -> Bluetooth -> D11 (Со значком с двумя экранами).
5.Скопируйте MAC адрес принтера: Панель управления -> Оборудование и звук -> Устройства и принтеры -> D11 -> Свойства -> Подключенное устройство -> Уникальный код.
6.Подключите устройство "MR-MARK-02" к компьютеру через кабель "microUSB - USB тип A" в порт "USB".
7.Запустите файл "start.bat" из корневой папки проекта.
8.Вставьте MAC адрес принтера из буфера обмена в поле снизу и нажмите кнопку "MAC".
9.Выберите COM порт с помощью списка в левом верхнем углу.
10.Подключите датчик "SMC-DS" к устройству "MR-MARK-02".
11.Нажмите кнопку "ТЕСТ", чтобы проверить как выглядит этикетка.
12.Измените длину датчика, если потребуется с помощью кнопки "Изменить длину".
13.Нажмите кнопку "ПЕЧАТЬ", чтобы отправить этикетку на печать в принтер.
14.После маркировки всех датчиков отправьте файл ids.txt из корневой папки проекта.
15.Удалите все идентификаторы в файле ids.txt из корневой папки проекта для следующего сеанса маркировки.
